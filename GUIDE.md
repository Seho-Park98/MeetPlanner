# MeetPlanner MCP 사용 가이드

## 1. 시작하기

### 사전 준비
1. Claude Desktop 설치
2. MeetPlanner MCP 설정 완료 (`claude_desktop_config.json`에 추가됨)
3. **Claude Desktop 재시작** (설정 변경 후 필수!)

### MCP 도구 확인
Claude Desktop 실행 후 채팅창 하단의 🔧 아이콘을 클릭하면 `recommend_meeting_place` 도구가 표시됩니다.

---

## 2. 사용 방법

### 기본 사용법
Claude에게 자연스럽게 만남 장소 추천을 요청하면 됩니다.

### 필수 정보
- **참가자 2명 이상**의 출발 위치
- 각 참가자의 **이름**과 **출발 장소**

### 선택 정보
- **만남 목적** (기본값: cafe_talk)

---

## 3. 채팅 예시

### 예시 1: 기본 추천 요청
```
A는 강남역에서, B는 홍대입구역에서 출발해.
카페에서 만나려고 하는데 어디가 좋을까?
```

### 예시 2: 3명 이상 모임
```
친구들이랑 저녁 약속을 잡으려고 해.
- 철수: 잠실역
- 영희: 신촌역
- 민수: 건대입구역
맛집이 많은 곳으로 추천해줘.
```

### 예시 3: 목적 명시
```
스터디 모임 장소를 찾고 있어.
참가자:
1. 김대리 - 역삼역
2. 이과장 - 여의도역
3. 박사원 - 성수역
조용히 공부할 수 있는 카페가 많은 곳이면 좋겠어.
```

### 예시 4: 간단한 요청
```
강남역이랑 홍대입구역 중간에서 만날 곳 추천해줘
```

### 예시 5: 데이트 장소
```
데이트 장소를 찾고 있어.
나는 압구정역에서 출발하고, 여자친구는 합정역에서 와.
분위기 좋은 곳으로 추천해줘!
```

### 예시 6: 지도로 보기 (VWorld 지도)
```
강남역이랑 홍대입구역 중간에서 만날 곳 추천해줘.
지도로 보여줘!
```
또는
```
철수는 잠실역, 영희는 신촌역에서 출발해.
맛집 추천해주고 지도로 표시해줘.
```

## 4. VWorld 지도 시각화

추천 결과를 VWorld 지도에서 시각적으로 확인할 수 있습니다.

### 지도 보기 요청 방법
- "지도로 보여줘"
- "지도에 표시해줘"
- "지도로 확인하고 싶어"

### 지도 기능
- **빨간 마커**: 추천 장소 (숫자 = 순위)
- **파란 마커**: 참가자 출발 위치
- **하단 패널**: 각 추천 장소의 상세 정보

### 지도 파일
- 요청 시 `map_result_날짜_시간.html` 파일이 생성됩니다
- 자동으로 브라우저에서 열립니다
- `C:\Users\tpgh0\Desktop\MeetPlanner\` 폴더에 저장됩니다

## 5. 지원하는 만남 목적

| 목적 | 키워드 예시 | 설명 |
|------|------------|------|
| `cafe_talk` | 카페, 수다, 대화 | 카페에서 대화하기 (기본값) |

## 6. 추천 결과 이해하기

### 결과 예시
```
## 추천 만남 장소

### 1위: 이태원역
- 위치: (37.5345, 126.9947)
- 예상 이동 시간: A: 25분, B: 30분
- 공정성 (표준편차): 3.5분
- 평균 이동 시간: 27.5분
- 추천 이유: 참가자들의 이동 시간 차이가 적당합니다...
```

### 결과 해석
- **순위**: 종합 점수가 높은 순서
- **예상 이동 시간**: 각 참가자가 해당 장소까지 대중교통으로 이동하는 예상 시간
- **공정성 (표준편차)**: 낮을수록 모두 비슷한 시간에 도착 (0에 가까울수록 공정)
- **평균 이동 시간**: 전체 참가자의 평균 이동 시간
- **추천 이유**: 왜 이 장소가 추천되었는지 설명

## 7. 지원하는 출발 위치

### 입력 가능한 형식
- **지하철역**: 강남역, 홍대입구역, 신촌역 등
- **주소**: 서울시 강남구 역삼동, 마포구 상수동 등
- **장소명**: 코엑스, 롯데월드타워, 서울역 등

### 주요 지원 지역
서울 전역의 주요 지하철역 50개소가 후보 장소로 등록되어 있습니다.

---

## 8. 팁

### 더 좋은 결과를 위한 팁
1. **출발 위치는 구체적으로**: "강남" 보다는 "강남역"이 정확합니다
2. **목적을 명시하면**: 해당 목적에 맞는 장소가 더 높은 점수를 받습니다
3. **참가자 이름 지정**: 결과에서 누가 얼마나 걸리는지 구분하기 쉽습니다

### 자주 묻는 질문
**Q: 왜 중간 지점이 아닌 다른 곳이 추천되나요?**
A: 단순 거리 중간이 아닌, 실제 이동 시간의 공정성과 장소의 목적 적합도를 함께 고려합니다.

**Q: 추천 장소가 마음에 안 들면?**
A: "다른 장소도 추천해줘" 또는 "좀 더 강남 쪽으로 추천해줘"라고 요청해보세요.

## 9. 문제 해결

### MCP 도구가 보이지 않을 때
1. Claude Desktop 완전 종료 후 재시작
2. `claude_desktop_config.json` 설정 확인
3. conda 환경 `PlayMCP`가 정상 설치되었는지 확인

### "주소를 찾을 수 없습니다" 오류
- 더 정확한 주소나 장소명으로 다시 시도
- 예: "홍대" → "홍대입구역"
